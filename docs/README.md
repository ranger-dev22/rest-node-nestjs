
## NestJS Boiler plate 

This boiler plate is build over  [Nest](https://github.com/nestjs/nest) framework in which we have covered few of the basic requirement which is required in majorly most of the project.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
